# My Project:
This is a simple task and class management app I built to help keep track of assignments, classes, and completed work all in one place. It started as a learning project while I was getting more comfortable with web development using Python and Flask. I am proud of what I've created but I do hope to continue working on it in the future and make it into exactly what I want it to be.

## What it does: 
In this app you can:
- Add and delete tasks
- Mark tasks as completed
- Organize tasks by class
- View active tasks separately from completed ones

## How to run it:
1. Clone the repository
2. Install the dependencies 
3. Run the app
4. Open the link that shows up in your terminal

## Programs I used:
- Python
- Flask
- HTML/CSS Files

# AI Reflection:
For this project, I used AI tools as a support system while building my app. While there were a lot of different aspects of the code that I already knew and understood, there was a bit of a learning curve that came along with learning how to use Flask. 
In the beginning, I mostly used AI support when I ran into errors or bugs in my code. For example, when something wasn't displaying correctly in my HTML pages, or when my Flask routes weren't working how I wanted them to be. I would ask for help understanding what might be wrong.
Instead of just giving me the final answer, I asked it to help me break down the problem so that I could figure out the issue and fully understand what was going wrong. This made debugging less frustrating and helped me understand how Flask and Python worked together.
I also used AI when I was unsure about what the exact structure of certain parts of my project should look like. For example, organizing my templates and separating different pages for tasks, completed items, and classes. It helped me with the general structure, and I adjusted things to fit my own vision for the app.
The biggest thing that I learned through this whole process was that AI is helpful, but it works best when you already have some understanding of what you're trying to do. When I used it to guide me and assist my problem-solving when I was stuck, things went much smoother than when I didn't know what I was doing. When I asked it to help me with things that I didn't fully understand yet, there were several times where my app ended up completely broken and I had to fix an even bigger mess than the one I had started with. 
Overall, I think AI was a useful learning tool in this project. It helped me understand concepts and gave support when it was needed.
